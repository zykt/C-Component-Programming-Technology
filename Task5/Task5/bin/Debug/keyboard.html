Keyboard events

Catching key presses

  1. Find either KeyDown or KeyUp event
  2. Create handler for them by double clicking or manually writing code
  3. Check which key launched event: if (e.KeyCode == Keys.Enter) ...

Notable Keycodes

Char keys: Keys.A, Keys.B ... Keys.Z

Number keys: Keys.D1, ... Keys.D0

System keys: Keys.Enter, Keys.Escape, Keys.Alt, Keys.Control ...

And many other

Example

private void Form1_KeyDown(object sender, KeyEventArgs e)
{
    if (e.KeyCode == Keys.Enter) 
    {
        this.Label1.Text = "Press any other key to fine their keycode";
    } else 
    {
        this.Label1.Text = $"Key pressed {e.KeyCode}";
    }
}
